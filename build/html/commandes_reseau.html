2. Commandes réseau

2.1 ip

Syntaxe :

ip <option>

Permet la manipulation et la configuration de l’adressage IP.

Exemple :

ip addr sh
ip route

Note

options connues :

  • L’option addr sh permet de lister les configurations réseaux des cartes Ethernet et Wifi présentes sur la machine.

  • L’option route permet d’afficher les routes.
